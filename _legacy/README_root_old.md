# Feathering

> 케이블 페더링 각도 계산/분석

## 개요

GeoView 탄성파 처리 소프트웨어.
Python 기반 GUI 애플리케이션으로, 케이블 페더링 각도 계산/분석 기능을 제공합니다.

## 기술 스택

- **언어**: Python
- **UI**: GUI (Tkinter)

## 설치 및 실행

### 사전 요구사항

- Python 3.8 이상

### 실행 방법

```bash
python main.py
```
